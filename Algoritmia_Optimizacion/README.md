# Algoritmia_Optimizacion

**Clonar Repositorio**
  * Elegir la ruta donde guardaremos los archivos del repositorio.
  * git clone https://github.com/santvz6/Algoritmia_Optimizacion.git
  * Introducimos user y contraseña: token
**Generar Token**
  GitHub > Settings > Developer Settings > Personal acces token

**Crear Archivo y Añadirlo**
  * Realizar lo siguiente dentro de la carpeta donde se guardó el repositorio
  * Crear fichero.py: nano prueba.py
    Para salir: Ctrl X, Sí y Enter
  * git add prueba.py
  * git commit o git commit -m "mensaje"
  * git push

**Copiar Repositorio con datos actualizados**
  * git pull (dentro de la carpeta donde clonamos el repositorio)

**Ver el estado del respositorio**
  * git status

**Eliminar un archivo**
  * git rm prueba.py
